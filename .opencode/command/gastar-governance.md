description: Validate that an OpenSpec change aligns with Gastar App durable docs before implementation
---

Use the `gastar-change-governance` skill.

For the target change:

- compare its artifacts with the relevant durable docs
- surface concrete mismatches or missing decisions
- say whether the change is ready for implementation
- point to the exact artifact that should be updated if it is not ready

Do not implement code from this command.
