Use the `gastar-project-intake` skill.

For the current request:

- classify the work type
- assess MVP fit
- identify affected domains
- identify durable docs or OpenSpec impact
- recommend the likely branch family
- recommend the next step: explore, propose, docs work, or apply an existing change

Keep the output short and decision-oriented.
